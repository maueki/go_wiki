# What's Irori

A simple and small knowledge sharing system like wiki.

# Requirements

* Go (1.4 or later)
* MongoDB
* coffee-script

# build & start

```
$ make
$ ./irori -bind :9090
```

Access to http://localhost:9090/wiki
